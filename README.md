## Animais Fantásticos

A front-end study project focused on building an interactive website about wild animals using semantic HTML, CSS, and JavaScript.

## About the Project
Animais Fantásticos is a fictional educational website created to practice website structure, content organization, and interactive front-end features.

The project presents information about different wild animals, including descriptions, facts, frequently asked questions, and contact sections, simulating a complete and professional multi-section webpage.

## Sections
- Navigation Menu
- Animal Information
- FAQ
- Contact
- Footer

## Technologies
- HTML  
- CSS  
- JavaScript

## About
This project was built for learning and practice purposes, focusing on semantic HTML structure, responsive layout organization, and DOM manipulation with JavaScript.
